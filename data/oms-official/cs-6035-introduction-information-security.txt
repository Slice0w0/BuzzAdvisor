Technical Requirements and Software
Important Information for CS 6035
A few notes on Virtual Machines (VMs):
Apple ARM-based SoCs are NOT supported in this course.
Due to technical limitations, only an Intel-based VM is provided.
You will be provided VMs in this class that were built using VirtualBox.
This course DOES NOT support or test with any other virtualization software.
